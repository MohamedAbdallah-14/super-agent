# templates/

This directory holds canonical artifact templates and examples.

Allowed:

- artifact templates
- schema-aligned examples
- documentation templates

Not allowed:

- live run outputs
- stale copied artifacts used as placeholders

Rules:

- Templates must match their schemas.
- Examples should be minimal, truthful, and runnable where applicable.
- Artifact templates under `templates/artifacts/` are the canonical starting point for phase outputs and handoffs.
