# skills/

This directory holds canonical reusable SuperAgent procedures.

Allowed:

- reusable operating procedures
- scoped execution guides
- review and verification skills

Not allowed:

- one-off task logs
- generated host exports

Rules:

- Keep skill behavior aligned with the canonical operating model.
- Remove superseded skills instead of leaving conflicting active instructions in place.
- Active skills currently expected in this directory:
  - `sa:brainstorming`
  - `sa:debugging`
  - `sa:design`
  - `sa:dispatching-parallel-agents`
  - `sa:executing-plans`
  - `sa:finishing-a-development-branch`
  - `sa:receiving-code-review`
  - `sa:requesting-code-review`
  - `sa:subagent-driven-development`
  - `sa:tdd`
  - `sa:using-git-worktrees`
  - `sa:using-skills`
  - `sa:verification`
  - `sa:writing-plans`
  - `sa:writing-skills`
  - `prepare-next`
  - `run-audit`
  - `scan-project`
  - `self-audit`
- Only active skills belong in this directory.
