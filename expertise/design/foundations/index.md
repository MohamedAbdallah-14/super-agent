# Directory Purpose

The `foundations` directory establishes the core visual building blocks required to create aesthetic, accessible, and balanced interfaces.

# Key Concepts

- Visual hierarchy and layout
- Accessible and inclusive design
- Typography, color, and spacing

# File Map

- `accessibility-design.md` — WCAG standards, contrast, and screen readers
- `color-theory.md` — palettes, contrast, and meaning
- `iconography.md` — standardizing icons and visual metaphors
- `motion-and-animation.md` — easing, timing, and purposeful motion
- `rtl-design.md` — right-to-left layout considerations
- `spacing-and-layout.md` — grids, margins, and the 8pt system
- `typography.md` — scale, line height, and font selection
- `visual-hierarchy.md` — guiding the user's eye

# Reading Guide

If setting up global styles → read `typography.md` and `color-theory.md`
If structuring a new screen → read `spacing-and-layout.md`
If auditing for inclusivity → read `accessibility-design.md`
